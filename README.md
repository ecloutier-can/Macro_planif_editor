# Macro_planif_editor
Créateur de macro planification.

## Description
Ce projet est un éditeur HTML simple pour la création et l'édition de macro planifications.

## Structure du projet
- `index.html` : Page principale de l'éditeur.
- `style.css` : Feuilles de style pour l'apparence.
- `script.js` : Scripts JavaScript pour la fonctionnalité.

## Comment utiliser
1. Ouvrez `index.html` dans un navigateur web.
2. Utilisez l'interface pour éditer votre macro planification.

## Développement
Pour développer, modifiez les fichiers HTML, CSS et JS selon vos besoins.
